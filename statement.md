## Problem Statement
Teachers often struggle to maintain manual attendance and mark registers. This project aims to digitize this process using a simple file-based management system.

## Scope
- **Target Users**: Teachers and Administrators.
- **Scope**: Local storage system for small to medium class sizes.

## High-Level Features
- CRUD Operations (Create, Read, Update, Delete).
- Input validation to prevent wrong data entry.
- Data persistence so records are not lost on exit.
