# Student Management System

## Overview
This is a Python-based CLI application to manage student records. It allows teachers to add, view, search, and delete student data. The data is stored persistently in a text file.

## Features
- Add new students with validation.
- View all student records in a table format.
- Search for students by Roll Number.
- Delete records permanently.
- Auto-save feature using file handling.

## How to Run
1. Ensure you have Python installed.
2. Run the command: `python main.py`
